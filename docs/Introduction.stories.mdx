# Welcome to Boilerplate

This is the available documentation for all components inside this project, in
an isolated scoped from this app's logic, data and context

In the left sidebar you can navigate between the available components

### Project Structure

Each component have inside component context - folder - yours `test` and
`stories` files

The component's structure was built inside the `Atomic Design` principles

- inside the `src/components` have the first 3 (three) concepts:
   - atoms
   - molecules
   - organisms

- templates and pages was stored each in your specific folders inside `src`

for more details about atomic design concepts access
[Atomic Design](https://bradfrost.com/blog/post/atomic-web-design/)
